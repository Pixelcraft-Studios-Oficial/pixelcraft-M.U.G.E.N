# M.U.G.E.N Engine

O M.U.G.E.N foi desenvolvido originalmente para DOS em linguagem C, utilizando o DJGPP e a Allegro — um excelente compilador e uma excelente biblioteca para jogos, respectivamente.
Atualmente, o M.U.G.E.N é desenvolvido para Microsoft Windows utilizando o Visual Studio Express e a SDL.

Por volta de 1997 ou 1998.  Nossa primeira versão de lançamento público foi
27/06/1999.
